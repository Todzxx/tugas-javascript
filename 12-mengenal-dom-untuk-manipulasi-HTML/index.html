Apa itu DOM API? dan Bagaimana Cara Menggunakannya?

DOM (Document Object Model) adalah antarmuka pemrograman untuk dokumen HTML dan XML. Ini merepresentasikan halaman sehingga program dapat mengubah struktur, gaya, dan konten dokumen.

1. Memilih Elemen (Selecting Elements)

Menggunakan metode seperti getElementById, querySelector, querySelectorAll.

Ini adalah paragraf.

2. Memanipulasi Konten (Manipulating Content)

Mengubah teks, HTML, atau atribut elemen.

Teks asli.

3. Menangani Event (Handling Events)

Menanggapi interaksi pengguna seperti klik, input, dll.

Belum ada event.

4. Membuat dan Menghapus Elemen (Creating & Removing Elements)

Menambahkan elemen baru ke DOM atau menghapus elemen yang sudah ada.
