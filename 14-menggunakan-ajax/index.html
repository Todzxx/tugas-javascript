Apa itu AJAX? dan Bagaimana Cara Menggunakannya?

AJAX (Asynchronous JavaScript and XML) adalah teknik pengembangan web untuk membuat aplikasi web asinkron. Dengan AJAX, aplikasi web dapat mengirim dan menerima data dari server secara asinkron (di latar belakang) tanpa mengganggu tampilan dan perilaku halaman yang ada.

Manfaat AJAX

  • Memperbarui halaman web tanpa memuat ulang seluruh halaman.
  • Membuat aplikasi web lebih cepat dan responsif.
  • Mengirim data ke server di latar belakang.

1. Menggunakan XMLHttpRequest (XHR)

Metode tradisional untuk membuat permintaan AJAX.

2. Menggunakan Fetch API

Antarmuka modern dan lebih kuat untuk membuat permintaan jaringan, mengembalikan Promise.
