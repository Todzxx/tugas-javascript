Memahami Struktur Data Array pada Javascript

Array adalah objek global yang digunakan untuk menyimpan daftar nilai. Nilai-nilai ini bisa berupa tipe data apa pun, dan diakses menggunakan indeks numerik.

Membuat Array

Cara paling umum untuk membuat array adalah dengan menggunakan literal array [].

Mengakses Elemen Array

Elemen diakses menggunakan indeks berbasis nol (indeks pertama adalah 0).

Properti .length

Properti ini mengembalikan jumlah elemen dalam array.

Metode Array Umum

Array memiliki banyak metode bawaan untuk manipulasi data.

Metode .push() & .pop()

Menambah elemen ke akhir array dan menghapus elemen dari akhir array.

Metode .map()

Membuat array baru dengan hasil dari memanggil fungsi pada setiap elemen array.

Metode .filter()

Membuat array baru dengan semua elemen yang lulus tes (fungsi callback).
