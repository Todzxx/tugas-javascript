Memahami 6 Bentuk Percabangan pada Javascript

Percabangan memungkinkan kita untuk menjalankan blok kode yang berbeda berdasarkan kondisi tertentu.

1. Percabangan if

Menjalankan blok kode jika kondisi bernilai true.

2. Percabangan if...else

Menjalankan satu blok kode jika kondisi true, dan blok lain jika false.

3. Percabangan if...else if...else

Memilih salah satu dari banyak blok kode untuk dieksekusi.

4. Percabangan switch

Alternatif dari if...else if...else untuk memilih blok kode berdasarkan nilai ekspresi.

5. Percabangan dengan Operator Ternary

Bentuk singkat dari if...else yang mengembalikan nilai.

6. Percabangan dengan Logical Operators (Short-circuiting)

Menggunakan operator && atau || untuk menjalankan kode secara kondisional.
