Enam Jenis Operator yang Wajib Kamu Ketahui di Javascript

Operator digunakan untuk melakukan operasi pada variabel dan nilai. Berikut adalah enam jenis operator utama di Javascript:

1. Operator Penugasan (Assignment Operators)

Digunakan untuk menetapkan nilai ke variabel.

2. Operator Aritmatika (Arithmetic Operators)

Digunakan untuk melakukan perhitungan aritmatika.

3. Operator Perbandingan (Comparison Operators)

Digunakan untuk membandingkan dua nilai.

4. Operator Logika (Logical Operators)

Digunakan untuk menentukan logika antara variabel atau nilai.

5. Operator Bitwise (Bitwise Operators)

Digunakan untuk melakukan operasi pada representasi biner angka.

6. Operator Ternary (Conditional Operator)

Operator kondisional yang menetapkan nilai ke variabel berdasarkan kondisi.
